# RTMX Homebrew Tap

This is the official Homebrew tap for RTMX CLI.

## Installation

```bash
brew tap rtmx-ai/tap
brew install rtmx
```

Or directly:

```bash
brew install rtmx-ai/tap/rtmx
```

## Updating

```bash
brew update
brew upgrade rtmx
```
